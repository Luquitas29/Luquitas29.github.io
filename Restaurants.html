Prediciendo el éxito de un restaurant¶

Lucas Berardo - Febrero 2022

Utilizando una base de reseas de restaurantes (del popular servicio Zomato) se puede generar una clasificacin automtica entre locales exitosos o no. En este notebook se muestra todo el proceso de exploracin, ingeniera de los atributos, seleccin de atributos y el entrenamiento del modelo. Para finalizar, se muestran las mtricas de evaluacin y se comparan diferentes algoritmos de clasificacin.
